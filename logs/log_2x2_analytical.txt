Графічний метод
КРОК 3: Графічне розв'язання гри 2x2

Матриця гри:
    B1   B2
A1 [7 3]
A2 [4 8]

Для гравця A (параметр p = P(A1)):
f1(p) = a21 + (a11-a21)*p = 4 + (3)*p
f2(p) = a22 + (a12-a22)*p = 8 + (-5)*p
A максимізує нижню огинаючу: g(p) = min(f1(p), f2(p))
Точка перетину f1=f2: p* = 0.5

Оптимум для A: p1 = 0.5, p2 = 0.5
Гарантований виграш A: v = 5.5

Для гравця B (параметр q = P(B1)):
u1(q) = a12 + (a11-a12)*q = 3 + (4)*q
u2(q) = a22 + (a21-a22)*q = 8 + (-4)*q
B мінімізує верхню огинаючу: h(q) = max(u1(q), u2(q))
Точка перетину u1=u2: q* = 0.625

Оптимум для B: q1 = 0.625, q2 = 0.375
Мінімізований виграш A: v = 5.5

Перевірка: p^T * A * q = 5.5
Ціна гри: v = 5.5

Аналітичний метод
КРОК 3: Аналітичне розв'язання гри 2x2

Матриця гри:
    B1   B2
A1 [7 3]
A2 [4 8]

Оптимальна змішана стратегія гравця A:
p1 = (a22 - a21) / (a11 - a21 - a12 + a22)
p1 = (8.0 - 4.0) / (7.0 - 4.0 - 3.0 + 8.0)
p1 = 4 / 8 = 0.5
p2 = 1 - p1 = 0.5

Оптимальна змішана стратегія гравця B:
q1 = (a22 - a12) / (a11 - a21 - a12 + a22)
q1 = (8 - 3) / 8 = 0.625
q2 = 1 - q1 = 0.375

Ціна гри:
v = (a11*a22 - a12*a21) / (a11 - a21 - a12 + a22)
v = (7*8 - 3*4) / 8
v = 44 / 8 = 5.5
